S03.s2 - Taller Integrador de JavaScript

Este taller integra conceptos básicos de JavaScript tales como variables, constantes, estructuras de control y manejo de cadenas, junto con operadores y funciones matemáticas. Los ejercicios están diseñados en contextos cotidianos

Ejercicio 1: Registro de Hotel

Introducción a variables, constantes y condicionales. Ingrese la cantidad de noches y el precio por noche para calcular el total. Se aplicará un descuento del 10% si se reservan 5 noches o más.

Ejercicio 2: Login Simple

Aplicación de estructuras de control y manejo de excepciones. Ingrese su usuario y contraseña para simular un login básico.

Ejercicio 3: Calculadora de Multiplicación

Uso de operadores aritméticos y función Math. Ingrese dos números para multiplicarlos y redondear el resultado.

Ejercicio 4: Manejo de Cadenas

Se aplican métodos de cadena: concatenación, obtención de longitud, extracción de subcadena y búsqueda de caracteres. Ingrese su nombre completo para generar un mensaje de bienvenida personalizado.
